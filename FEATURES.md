Basic Features for a Mood Logger

Core Functionality:

- Mood Entry: Log mood on a scale (1-10) or with predefined options (happy, sad, anxious, etc.)
- Notes/Journal: Add context to each entry with free-text notes
- Timestamps: Automatic date/time recording
- Tags/Categories: Label entries (work, personal, health, etc.)

Data & Insights:

- History View: Browse past entries by date range
- Mood Trends: Visualize patterns over time (charts/graphs)
- Statistics: Average mood, most common tags, streaks

User Experience:

- Quick Log: Fast mood entry without friction
- Reminders: Optional prompts to log daily
- Export: Download data (CSV, JSON)
- Search/Filter: Find entries by date, tag, or keyword

Data Management:

- Local Storage: Persist entries on device
- Backup/Sync: Optional cloud backup
- Privacy Controls: Data stays on user's device
